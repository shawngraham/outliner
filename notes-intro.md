# Welcome to Note Cards
This outlining tool helps you organize your thoughts visually with connected cards.
#-------
# Getting Started
1. Double-click to create new cards
2. Drag cards to reposition
3. Connect related ideas
#-------
# Key Features
- Expandable cards for detailed content
- Visual connections between ideas
- Persistent storage of your work
#-------
# Tips for Success
Start with broad concepts, then add supporting details. Use connections to show relationships between ideas.
